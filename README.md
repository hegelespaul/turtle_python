# Generación Geometrica y Arte Computacional con Python (Turtle)

Este repositorio reúne una serie de exploraciones en programación creativa orientadas a la enseñanza de conceptos matemáticos y computacionales mediante el uso de Python y el módulo Turtle.

El proyecto se sitúa en la intersección entre arte generativo, geometría, sistemas complejos y pedagogía, proponiendo una aproximación accesible a la construcción de imágenes a partir de código.

---

## Organización de archivos

Los contenidos del repositorio se estructuran de la siguiente manera:

1. **Geometría Analítica Bidimensional:** archivos 01 a 05  
2. **Trigonometría y Ondas:** archivos 06 a 08  
3. **Sistemas Probabilísticos:** archivo 09  
4. **Recursividad y Fractales:** archivos 10, 11 y 12  
5. **Sistemas Complejos:** archivo 13  

---

## Objetivo

- Explorar el potencial de herramientas simples como Turtle en contextos contemporáneos  
- Facilitar la visualización de conceptos matemáticos  
- Servir como introducción a la programación creativa  
- Proponer un puente entre arte, matemáticas y computación  

---

## Alcance

Los códigos aquí presentados corresponden a experiencias exploratorias y no a una implementación pedagógica formal consolidada.

Se trata de aproximaciones iniciales que buscan evidenciar el potencial del enfoque, más que ofrecer soluciones definitivas.

---
